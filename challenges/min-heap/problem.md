N개의 연산이 주어져요.

- 연산값이 양의 정수 x이면, x를 자료 구조에 넣어요.
- 연산값이 0이면, 자료 구조에서 가장 작은 값을 꺼내 출력해요. 비어 있으면 0을 출력해요.

### 입력

첫 번째 줄에 연산의 개수 N이 주어져요.

이후 N개의 줄에 연산값이 하나씩 주어져요.

- $1 \leq N \leq 100{,}000$
- 연산값은 $0$ 이상 $2^{31} - 1$ 미만의 정수예요.

### 출력

0 연산이 주어질 때마다 결과를 한 줄씩 출력하세요.
