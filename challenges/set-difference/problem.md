두 집합 A, B가 주어져요. A에는 있지만 B에는 없는 원소들을 구해보세요.

### 입력

첫 번째 줄에 집합 A의 원소 개수 N, 집합 B의 원소 개수 M이 공백으로 구분되어 주어져요.

두 번째 줄에 집합 A의 원소 N개가 공백으로 구분되어 주어져요.

세 번째 줄에 집합 B의 원소 M개가 공백으로 구분되어 주어져요.

- $1 \leq N, M \leq 500{,}000$
- 각 원소는 $1$ 이상 $10^8$ 이하의 정수예요.
- 집합 내 원소는 중복되지 않아요.

### 출력

첫 번째 줄에 차집합의 원소 개수를 출력하세요.

원소가 있다면 두 번째 줄에 오름차순으로 공백으로 구분하여 출력하세요.
