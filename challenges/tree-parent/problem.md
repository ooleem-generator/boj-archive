N개의 노드로 이루어진 트리가 주어져요. 루트 노드는 1번이에요.

각 노드의 부모 노드를 구해보세요.

### 입력

첫 번째 줄에 노드의 수 N이 주어져요.

이후 N-1개의 줄에 트리의 간선이 하나씩 주어져요.

- $2 \leq N \leq 100{,}000$

### 출력

2번 노드부터 N번 노드까지 각 노드의 부모 노드를 한 줄씩 출력하세요.
