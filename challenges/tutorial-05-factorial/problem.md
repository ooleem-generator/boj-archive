정수 N이 주어집니다. N!을 출력하세요.

$$N! = N \times (N-1) \times \cdots \times 1$$

단, $0! = 1$로 정의합니다.

## 입력

정수 N (0 ≤ N ≤ 12)이 주어집니다.

## 출력

N!을 출력합니다.
