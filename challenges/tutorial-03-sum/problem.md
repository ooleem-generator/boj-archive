N개의 정수가 주어집니다. 모두 더한 값을 출력하세요.

샘플 케이스를 모두 통과하면 제출 버튼이 활성화됩니다. 제출 전에 세 케이스를 전부 확인해보세요.

## 입력

첫째 줄에 정수 N (1 ≤ N ≤ 100)이 주어집니다.

둘째 줄에 N개의 정수 $A_i$ (1 ≤ $A_i$ ≤ 1,000)가 공백으로 구분되어 주어집니다.

## 출력

N개의 정수의 합을 출력합니다.
