정수 N을 1, 2, 3의 합으로 나타내는 방법의 수를 구해보세요. 순서가 다른 경우는 같은 방법으로 봐요.

예를 들어 4를 나타내는 방법은 1+1+1+1, 1+1+2, 1+3, 2+2로 4가지예요.

테스트 케이스가 여러 개 주어져요.

### 입력

첫 번째 줄에 테스트 케이스의 수 T가 주어져요.

이후 T개의 줄에 N이 하나씩 주어져요.

- $1 \leq T \leq 1{,}000$
- $1 \leq N \leq 10{,}000$

### 출력

각 테스트 케이스마다 방법의 수를 한 줄씩 출력하세요.
