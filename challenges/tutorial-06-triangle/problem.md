세 양의 정수 A, B, C가 주어집니다. 이 세 수를 세 변의 길이로 하는 직각삼각형이 존재하는지 판별하세요.

NEXT JUDGE의 모든 문제는 GitHub PR을 통해 누구나 기여할 수 있습니다. 이 플랫폼의 문제들은 알고리즘을 공부하는 사람들이 직접 만들어 나갑니다.

## 입력

세 양의 정수 A, B, C (1 ≤ A, B, C ≤ 1,000)가 한 줄에 공백으로 구분되어 주어집니다.

## 출력

직각삼각형이 존재하면 `YES`, 아니면 `NO`를 출력합니다.
