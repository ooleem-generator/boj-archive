N개의 정수 중 가장 작은 수를 출력하세요.

이 문제부터는 **히든 테스트케이스**가 있습니다. 샘플을 모두 통과한 뒤 제출하면, 공개되지 않은 추가 케이스로 최종 채점이 이루어집니다.


## 입력

첫째 줄에 정수 N (1 ≤ N ≤ 100)이 주어집니다.

둘째 줄에 N개의 정수 $A_i$ (-100 ≤ $A_i$ ≤ 100)가 공백으로 구분되어 주어집니다.

## 출력

N개의 정수 중 가장 작은 값을 출력합니다.
