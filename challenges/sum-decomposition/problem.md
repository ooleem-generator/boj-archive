0부터 N까지의 정수 중 K개를 골라 더했을 때 합이 N이 되는 경우의 수를 구해보세요.

같은 수를 여러 번 고를 수 있고, 순서가 다르면 다른 경우로 취급해요.

### 입력

정수 N과 K가 공백으로 구분되어 주어져요.

- $1 \leq N \leq 200$
- $1 \leq K \leq 200$

### 출력

경우의 수를 $1{,}000{,}000{,}000$으로 나눈 나머지를 출력하세요.
